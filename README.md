# 图书管理系统 java1188

## 项目概述

图书管理系统是一款基于SpringBoot和Vue的前后端分离项目，主要应用于图书馆的图书信息管理、用户借阅管理等方面。

## 技术栈

- 开发环境：IDEA、JDK1.8、Maven、Mysql8.0、Node16
- 前端技术：Vue
- 后端技术：SpringBoot

## 功能模块

### 系统角色

- 普通用户
- 管理员

### 主要功能

- 图书信息管理
  - 图书信息的增删改查
- 图书类别管理
  - 类别的增删改查
- 借阅信息管理
  - 借阅信息的查看和统计
- 用户管理
  - 用户信息的增删改查
- 修改密码
- 用户借书
- 用户还书

## 系统运行环境

- 后端：Java 1.8+，Mysql 8.0+
- 前端：现代浏览器支持

## 目录结构

```
项目目录
│
├── 后端目录（SpringBoot项目）
└── 前端目录（Vue项目）
```

## 核心亮点

- 前后端分离，便于维护和扩展
- 使用SpringBoot和Vue，保证系统的高效和稳定性
- 清晰的角色权限划分，满足不同用户的操作需求
- 完善的图书和借阅信息管理，提高图书馆工作效率
